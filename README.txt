	Features

	v1

		Copies all files with a specificed extension from a directory to a destination folder, optionally including subfolders.

		Searches directory, optionally including subfolders, for files with specificed substring in the file name and copies them to a destination folder
		
		Copies folders containing a substring in folder name.

		Moves files into new folders named according to the first (n) characters in the file name.

		Moves files into new folders named according to the file's date modified 

		Always checks for duplicate files. If true, creates a new name for the file and copies it.

		Executable program with an easy to use GUI
